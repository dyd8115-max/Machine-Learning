# 기계학습 — 1. 학습이란 무엇인가

## 1. 데이터로부터 학습하기

강아지를 생물학적 정의로 배우는 게 아니라, 강아지 사진을 보며 배우듯  
**기계도 규칙을 직접 코딩하는 게 아니라 데이터에서 패턴을 스스로 찾는다.**

> 데이터 $(x_1, \ldots, x_n)$ 와 결과 $(y_1, \ldots, y_n)$ 사이의 관계를 가장 잘 나타내는 함수 $f$ 를 찾는 과정

---

## 2. 문제의 구조화 (Problem Setup)

### 입출력 공간

| | 기호 | 설명 |
|---|---|---|
| 입력 공간 | $X = \mathbb{R}^d$ | d차원 벡터 $\mathbf{x} = (x_1, x_2, \ldots, x_d)$ |
| 출력 공간 | $Y = \{+1, -1\}$ | 최종 결정 (이진 분류 기준) |

### 학습의 구성 요소

| 구성 요소 | 기호 | 대출 심사 예시 |
|---|---|---|
| 입력 | $x$ | 나이, 소득, 부채 등 |
| 출력 | $y$ | 승인 (+1) / 거부 (−1) |
| 목표 함수 | $f: X \to Y$ | 이상적인 승인 공식 (**알 수 없음**) |
| 학습 데이터 | $(x_i, y_i)$ | 과거 심사 기록 |
| 가설 집합 | $H$ | 후보 공식들의 집합 |
| 최종 가설 | $g: X \to Y$ | 학습으로 얻은 공식 |

### 핵심 구조

```
미지의 목표 함수 f
        ↓
  학습 데이터 (xᵢ, yᵢ) ──→ 학습 알고리즘 A ──→ 최종 가설 g ≈ f
                                  ↑
                           가설 집합 H
```

- **학습 모델** = 가설 집합 $H$ + 학습 알고리즘 $A$
- 목표: $g$ 가 $f$ 에 최대한 근접하도록

---

## 3. 학습의 유형 (Types of Learning)

### 세 가지 패러다임 비교

| | 지도 학습 | 비지도 학습 | 강화 학습 |
|---|---|---|---|
| 학습 데이터 | $(x,\ y_{\text{정답}})$ | $(x)$ | $(x,\ y,\ \text{보상})$ |
| 피드백 | 즉각적 정답 | 없음 | 지연된 보상 |
| 목표 | 함수 근사 | 구조/패턴 발견 | 누적 보상 최대화 |
| 예시 | 스팸 분류, 집값 예측 | 군집화, 차원 축소 | AlphaGo, 자율주행 |

### 지도 학습 (Supervised Learning)

각 입력에 **정답 레이블**이 주어진 학습 방식

- **분류 (Classification)** — 출력이 이산값 (클래스)
  - 예) 스팸 여부, 이미지 분류
  - 모델: Perceptron, SVM, 신경망
- **회귀 (Regression)** — 출력이 연속값
  - 예) 주택 가격 예측, 수요 예측
  - 모델: 선형 회귀, 신경망

**전형적인 절차**

```
데이터 수집 → 전처리 → 특징 추출 → 모델 학습 → 모델 평가 → 배포
```

### 비지도 학습 (Unsupervised Learning)

레이블 없이 데이터의 내재적 구조를 자율적으로 파악

| 기법 | 알고리즘 |
|---|---|
| 군집화 (Clustering) | k-means, GMM |
| 차원 축소 (Dim. Reduction) | PCA, t-SNE |
| 밀도 추정 (Density Est.) | KDE, GMM |

> 지도 학습의 **전처리 단계**로도 활용됨

### 강화 학습 (Reinforcement Learning)

정답이 아닌 **보상/페널티 신호**를 통해 최적 정책(policy)을 학습

- 순차적 의사결정(sequential decision-making)에 강점
- 응용: 게임 AI (AlphaGo, DQN), 로봇 제어, 자율주행
