# 기계학습 — 1. 학습이란 무엇인가

## 학습의 정의

규칙을 직접 코딩하는 게 아니라, **데이터로부터 패턴을 찾는 것**

- 데이터 사이에 관계는 있지만 정확히 뭔지 모를 때
- 그 관계를 근사(approximation)하는 함수 $f$ 를 찾는 과정

---

## 문제 구조 (Problem Setup)

| 구성 요소 | 기호 | 설명 |
|---|---|---|
| 입력 | $\mathbf{x} \in \mathbb{R}^d$ | 나이, 소득, 부채 등 특징 벡터 |
| 출력 | $y \in \{+1, -1\}$ | 최종 결정값 |
| 목표 함수 | $f: X \to Y$ | 이상적인 정답 함수 — **알 수 없음** |
| 학습 데이터 | $(x_i, y_i)$ | 과거 기록 |
| 가설 집합 | $H$ | 후보 함수들의 집합 |
| 최종 가설 | $g \approx f$ | 학습으로 얻은 함수 |

**학습 모델** = $H$ (가설 집합) + $A$ (학습 알고리즘) → 우리가 선택하는 도구

```
f (미지)
   ↓
학습 데이터 + H → 알고리즘 A → g ≈ f
```

---

## 학습의 유형

### 지도 학습 (Supervised Learning)
입력 + **정답 레이블** 쌍으로 학습

- **분류 (Classification)** — 출력이 이산값 → 스팸 탐지, 이미지 분류
- **회귀 (Regression)** — 출력이 연속값 → 집값 예측, 수요 예측

### 비지도 학습 (Unsupervised Learning)
레이블 없이 **데이터 구조/패턴** 탐색

- 군집화: k-means, GMM
- 차원 축소: PCA, t-SNE

### 강화 학습 (Reinforcement Learning)
**보상/페널티** 신호로 최적 정책(policy) 학습

- 피드백이 즉각적이지 않고 지연됨
- 순차적 의사결정에 강점 → AlphaGo, 자율주행

---

## 세 유형 비교

| | 지도 학습 | 비지도 학습 | 강화 학습 |
|---|---|---|---|
| 데이터 | $(x, y)$ | $(x)$ | $(x, y, \text{보상})$ |
| 피드백 | 즉각적 정답 | 없음 | 지연된 보상 |
| 목표 | $g \approx f$ | 패턴 발견 | 누적 보상 최대화 |
