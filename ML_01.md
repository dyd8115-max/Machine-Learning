# 기계학습 — 2. 선형대수학

## 1. 벡터 이론

### 기본 개념

- **벡터**: 크기와 방향을 가지는 물리량
- **특징 벡터 (Feature Vector)**: 어떤 대상을 수치로 표현한 벡터
  - 예) 사람을 키·몸무게·나이로 표현 → $x = [175, 70, 23]$
  - ML 모델은 이 벡터를 입력으로 받아 학습
- **열벡터 (Column Vector)**: 벡터를 세로(d×1 행렬)로 표현하는 방식
  - ML에서는 관례적으로 열벡터를 기본으로 사용
  - 가로로 쓸 땐 전치 기호를 붙여 $x^\top$ 으로 표현
  - $\mathbf{x} = [x_1, x_2, \ldots, x_d]^\top \in \mathbb{R}^d$

| 개념 | 정의 | 수식 |
|---|---|---|
| 전치 (Transpose) | 행과 열을 바꿈 | $x^\top = [x_1, x_2, \ldots, x_N]$ |
| 크기 (Norm) | 원점에서의 거리 | $\|x\| = \sqrt{x^\top x}$ |
| 단위벡터 (Unit Vector) | 크기가 1인 벡터 | $u = v / \|v\|$ |

> 벡터를 단위벡터로 만드는 과정 = **정규화(normalization)**

---

### 내적 / 외적

**내적 (Dot Product)**

$$A \cdot B = A_1B_1 + A_2B_2 + \ldots + A_nB_n = A^\top B \quad \text{(결과: 스칼라)}$$

- 각도 θ가 주어질 때: $A \cdot B = \|A\|\|B\|\cos\theta$
- θ = 90° → 내적 = 0 → **두 벡터가 직교(orthogonal)**
- 기계학습 활용: **유사도 측정, 투영, Attention**

**외적 (Cross Product)** — 3D 전용

$$\|A \times B\| = \|A\|\|B\|\sin\theta \quad \text{(결과: 벡터)}$$

- A와 B에 동시에 수직인 벡터 (오른손 법칙)
- 기계학습 활용: 법선벡터, 3D 회전

| 성질 | 내적 | 외적 |
|---|---|---|
| 결과 | 스칼라 | 벡터 (3D 전용) |
| 교환법칙 | $A \cdot B = B \cdot A$ ✓ | $A \times B = -(B \times A)$ ✗ |
| 자기 곱 | $A \cdot A = \|A\|^2$ | $A \times A = 0$ |
| 기하학적 의미 | cosθ, 투영/유사도 | sinθ, 면적/수직 방향 |

---

### 선형 결합 / 선형 종속 / 선형 독립

- **선형 결합**: $y = \alpha_1 x_1 + \alpha_2 x_2 + \ldots + \alpha_m x_m$
- **선형 종속**: 집합 내 하나의 벡터가 나머지의 선형 결합으로 표현 가능 → **중복 정보 존재**
- **선형 독립**: 어떤 벡터도 나머지의 선형 결합으로 표현 불가 → **독립적인 정보**

---

### 기저 집합 (Basis Set)

- N×1 모든 벡터를 나타낼 수 있는 벡터의 집합
- 기저 조건: **선형 독립**
- $x = a_1u_1 + \ldots + a_nu_n$ (계수 $a_i$: 벡터 x의 성분)

| 종류 | 조건 | 특징 |
|---|---|---|
| 직교 기저 (Orthogonal) | $u_i \cdot u_j = 0\ (i \neq j)$ | 기저 벡터들이 서로 수직 |
| 정규직교 기저 (Orthonormal) | $u_i \cdot u_j = \delta_{ij}$ | 단위벡터 + 서로 직교 → 투영 계산 단순화: $a_i = x \cdot u_i$ |

> 임의의 선형 독립 집합 → **Gram-Schmidt** 과정으로 정규직교 기저로 변환 가능

**기계학습 의미**: 신경망 각 층이 데이터를 새로운 기저로 표현 (representation learning)  
→ 좋은 기저(특징)를 찾는 것이 딥러닝의 핵심 목표 중 하나

---

### Span / 차원 확대·축소 / 투영

**Span (생성)**

- 벡터 집합 $\{v_1, \ldots, v_k\}$ 의 모든 가능한 선형 결합들의 집합
- 예) $v_1=[1,0]^\top,\ v_2=[0,1]^\top$ → span = $\mathbb{R}^2$

**차원 확대 (Embedding)**: 저차원 → 고차원으로 새로운 span 생성 → 선형 분리 가능성 증가
- 예) SVM 커널 트릭, 신경망 은닉층

**차원 축소 (Dimensionality Reduction)**: 고차원 → 핵심 정보가 span하는 저차원으로 투영
- 노이즈·중복 특징 제거 → 계산 효율화 및 일반화 성능 향상

**투영 (Projection)**

- 벡터 $x$ 를 단위벡터 $u$ 방향으로 투영: $(x \cdot u) \cdot u$
- 부분공간 $S = \text{span}(\{u_1,\ldots,u_k\})$ 로의 투영: $\text{proj}(x) = UU^\top x$
- 잔차 $r = x - \text{proj}(x)$: S로 설명 못하는 나머지 성분 (⊥S)

> **선형 변환 = 새로운 벡터 공간으로의 투영**  
> $y = Wx$ → x의 각 성분이 W의 행벡터 방향으로 투영된 새 벡터  
> Deep Learning: 각 층의 가중치 행렬 W가 데이터를 새로운 부분공간으로 반복 투영·재구성

---

### 선형 종속과 표현의 축소 / Latent Representation

- 선형 종속 벡터 집합: rank < 벡터 수 → 실제 span 차원은 더 낮음 → **중복 정보 존재**
- d차원 특징 벡터가 실제로는 k차원 부분공간에 집중 (k << d) → k개의 기저로 압축 표현 가능

**Deep Learning의 Latent Representation**

- 은닉층(hidden layer) 출력 = 입력 x의 새로운 벡터 공간 좌표 = **Latent Vector z**
- z가 span하는 공간 = **Latent Space**: 의미적으로 유사한 데이터가 가까이 위치
- **Autoencoder**: Encoder $x \to z$ (차원 축소) / Decoder $z \to x'$ (차원 복원)
- **Transformer**: Attention이 입력 벡터들의 가중 선형 결합 → 새로운 부분공간으로 context 표현

## 2. 행렬 대수

### 특수 행렬

| 종류 | 조건 | 비고 |
|---|---|---|
| 정방 행렬 | n×n | 역행렬, trace 등 정의 가능 |
| 대각 행렬 | 대각 성분 외 모두 0 | $D = \text{diag}(d_1, \ldots, d_n)$ |
| 항등 행렬 | 대각=1, 나머지=0 | $AI = IA = A$ |
| 대칭 행렬 | $A = A^\top$ | 공분산 행렬, 그람 행렬 ($G^\top G$) |
| 직교 행렬 | $A^\top A = AA^\top = I$ | $A^{-1} = A^\top$ → 계산 효율 높음 |

> 직교 행렬은 벡터의 **길이와 내적(각도) 보존** → 회전 변환

---

### 행렬 곱셈

A (m×k), B (k×n) → C = AB는 m×n 행렬

$$C_{ij} = \sum_l A_{il} B_{lj} \quad \text{(i행·j열 = A의 i번째 행벡터와 B의 j번째 열벡터의 내적)}$$

- 결합법칙: $(AB)C = A(BC)$ ✓
- 교환법칙: $AB \neq BA$ (일반적으로)
- $(AB)^\top = B^\top A^\top$

---

### Rank

- 행렬 A에서 **선형 독립인 열벡터(또는 행벡터)의 최대 수**
- $\text{rank}(A) \leq \min(m, n)$

| 조건 | 명칭 | 의미 |
|---|---|---|
| $\text{rank}(A_{n \times n}) = n$ | 비특이(non-singular) | 역행렬 존재 |
| $\text{rank}(A_{n \times n}) < n$ | 특이(singular) | 역행렬 없음 |

**기계학습 의미**: 데이터 행렬 X의 rank = 데이터가 실제로 span하는 공간의 차원  
rank가 낮으면 선형 종속 특징이 많아 중복 정보 존재 → 차원 축소 여지 있음

---

### 역행렬 (Inverse Matrix)

- $AX = XA = I$ 를 만족하는 $X = A^{-1}$
- 존재 조건: $\det(A) \neq 0$ (비특이 행렬)

주요 성질:
- $(A^{-1})^{-1} = A$
- $(AB)^{-1} = B^{-1}A^{-1}$
- $(A^\top)^{-1} = (A^{-1})^\top$

**기계학습 활용**
- 선형 회귀 정규방정식: $\theta = (X^\top X)^{-1} X^\top y$
  - $X^\top X$ 가 특이행렬이면 역행렬 없음 → **의사역행렬(pseudo-inverse)** 사용
- 직교 행렬: $A^{-1} = A^\top$ → 역행렬 계산 불필요

---

### 선형 변환 (Linear Transformation)

- 벡터 공간 $X^N$ → $Y^M$ 으로 매핑: $y = Wx$ (W: M×N 행렬)
- M < N: 차원 축소 (압축 표현) / M > N: 차원 확대 (임베딩)

**신경망과 선형 변환**

각 완전연결층(FC layer): $h = \sigma(Wx + b)$
- W: 선형 변환, σ: 비선형 활성화
- 층을 쌓을수록 데이터를 점차 새로운 부분공간으로 반복 변환 → 복잡한 표현 학습

---

## 핵심 개념 — 딥러닝 연결

| 개념 | 핵심 | 딥러닝 연결 |
|---|---|---|
| Span | 벡터들로 도달 가능한 공간 전체 | Latent Space의 구조 정의 |
| 선형 독립 | 어떤 벡터도 나머지의 선형 결합으로 표현 불가 | 중복 없는 정보 방향 |
| 투영 | 부분공간으로의 최근접 표현 ($UU^\top x$) | 각 층의 선형 변환 (Wx) |
| Rank | 선형 독립 벡터의 최대 수 = span 차원 | 가중치 행렬의 유효 차원 |
| 차원 확대/축소 | Embedding / Dimensionality Reduction | Encoder-Decoder, Attention |
| Latent Vector z | x의 새로운 벡터 공간 좌표 (선형 종속 제거) | Autoencoder, Transformer |
