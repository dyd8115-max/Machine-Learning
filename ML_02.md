# 기계학습 — 3. 확률

## 1. 확률 공간의 수학적 구조

### 1-1. 확률이란?

- **확률 (Probability)**: 불확실한 사건(event)에 수치 [0,1]를 부여하는 함수

| 관점 | 정의 | 예시 | 한계 |
|---|---|---|---|
| 빈도주의 (Frequentist) | 실험을 무한히 반복할 때 A가 일어나는 상대 빈도 | 동전 10,000번 → 앞면 5,023번 → P(앞) ≈ 0.5023 | 반복 불가 사건에 적용 어려움 |
| 베이지안 (Bayesian) | 사건 A에 대한 주관적 믿음의 정도 (degree of belief) | '이 환자가 독감일 확률'처럼 사전 지식 반영 가능 | - |

---

### 1-2. 확률 공간 (Probability Space) = (Ω, F, P) (시험x)

| 기호 | 이름 | 정의 |
|---|---|---|
| $\Omega$ | 표본 공간 (Sample Space) | 가능한 모든 결과의 집합 |
| $F$ | σ-대수 (σ-algebra) | 측정 가능한 사건들의 집합 |
| $P$ | 확률 측도 (Probability Measure) | F의 각 사건에 [0,1] 실수를 부여하는 함수 |

**예시 — 동전 2회 던지기**

| $\Omega$ | $F$ (사건의 예) | $P$ (확률 값) |
|---|---|---|
| {HH, HT, TH, TT} | '앞면이 1번 이상' = {HH,HT,TH} | P({HH}) = 1/4 |
| | '앞면이 없음' = {TT} | P(앞면 1번 이상) = 3/4 |

---

### 1-3. 표본 공간 (Ω) 과 사건 (Event)

**표본 공간 Ω**

- 확률 실험에서 나올 수 있는 모든 결과(outcome)의 집합
- $\omega$ (소문자) = **근원 사건(elementary event)** → 실험에서 나오는 개별 결과 하나
- $\Omega$ (대문자) = **표본 공간** → 모든 근원 사건 $\omega$ 들의 집합 → $\omega \in \Omega$
- 동전: $\Omega = \{H, T\}$ / 주사위: $\Omega = \{1,2,3,4,5,6\}$ / 온도계: $\Omega = \mathbb{R}$

**사건 $A \in F$**

- 사건은 표본 공간의 **부분집합** $A \subseteq \Omega$
- $A \in F$ = "A는 확률을 계산할 수 있는 사건이다"
- 전체 사건 $\Omega$ (반드시 일어남) / 공사건 $\emptyset$ (절대 불가)

예시) 동전 2회, $\Omega = \{HH, HT, TH, TT\}$
- $A = \{HH, HT\}$ → $A \in F$ → $P(A)$ 계산 가능
- $A = \{HH\}$ → $A \in F$ → $P(A) = 1/4$

**개념 정리**

| 기호 | 이름 | 설명 | 예시 (주사위) |
|---|---|---|---|
| $\omega$ | 근원 사건 | 실험 결과 하나 | $\omega = 3$ |
| $\Omega$ | 표본 공간 | 모든 $\omega$ 의 집합 | $\Omega = \{1,2,3,4,5,6\}$ |
| $A$ | 사건 | $\omega$ 들의 묶음 ($\Omega$ 의 부분집합) | $A = \{2,4,6\}$ (짝수) |
| $F$ | σ-대수 | 확률을 부여할 수 있는 사건들의 모음 | $\{\emptyset, \{짝수\}, \{홀수\}, \Omega, \ldots\}$ |

> $\omega \in \Omega$ (근원 사건은 표본 공간의 원소), $A \subseteq \Omega$ (사건은 표본 공간의 부분집합), $A \in F$ (사건은 σ-대수의 원소)

**사건의 집합 연산**

| 기호 | 이름 | 의미 |
|---|---|---|
| $A \cup B$ | 합사건 (Union) | A 또는 B가 일어나는 사건 |
| $A \cap B$ | 곱사건 (Intersection) | A와 B가 동시에 일어나는 사건 |
| $A^c$ | 여사건 (Complement) | A가 일어나지 않는 사건, $P(A^c) = 1 - P(A)$ |
| $A \cap B = \emptyset$ | 배반사건 (Mutually Exclusive) | A와 B가 동시에 일어날 수 없음 |

---

### 1-4. σ-대수 (σ-algebra) F

- "어떤 사건에 확률을 부여할 수 있는가?"를 결정하는 수학적 구조
- 확률을 계산하는 함수 P가 입력받을 수 있는 **유효한 사건들의 집합**

| 경우 | 설명 | 예시 |
|---|---|---|
| $F = 2^\Omega$ | $\Omega$가 유한/자연수/정수/유리수 집합 → 가능한 모든 부분집합에 확률 부여 가능 | 동전: $\Omega=\{H,T\}$ → $F = \{\emptyset, \{H\}, \{T\}, \{H,T\}\}$ |
| $F \subseteq 2^\Omega$ | $\Omega = \mathbb{R}$ 처럼 비가산 무한 집합 → 모든 부분집합에 일관된 확률 부여 수학적으로 불가 → 확률을 줄 수 있는 사건만 골라 담은 것 | 온도: $\Omega = \mathbb{R}$ → 구간 $[a,b]$ 같은 사건에만 확률 부여 |

> 어떤 경우든 $\emptyset \in F,\ \Omega \in F$ 는 항상 성립 → $P(\emptyset) = 0,\ P(\Omega) = 1$

---

### 1-5. 확률 측도 P & Kolmogorov 공리 (1933)

```math
P : F \to [0,1] \quad \text{사건(F의 원소)에 실수 값을 부여하는 함수}
```

| 공리 | 이름 | 수식 | 의미 |
|---|---|---|---|
| 공리 1 | 비음수성 (Non-negativity) | $P(A) \geq 0$ | 확률은 음수가 될 수 없다 |
| 공리 2 | 규범성 (Normalization) | $P(\Omega) = 1$ | 전체 표본 공간의 확률은 반드시 1 |
| 공리 3 | 가산 가법성 (σ-additivity) | $A_i \cap A_j = \emptyset$ 이면 $P(A_1 \cup A_2 \cup \cdots) = P(A_1) + P(A_2) + \cdots$ | 서로 배반인 사건들의 합사건 확률 = 각 확률의 합 |

**공리로부터 유도되는 중요 성질**

- $P(\emptyset) = 0$ (불가능한 사건)
- $P(A^c) = 1 - P(A)$ (여사건)
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ (포함-배제)

## 2. 확률변수와 분포

### 2-1. 확률변수 (Random Variable)

```math
X : \Omega \to \mathbb{R} \quad \text{사건 집합을 실수로 매핑하는 함수}
```

> 확률변수 ≠ 변수, **확률변수 = 함수**

- 이름이 '변수'지만 실제로는 $\Omega \to \mathbb{R}$ 방향의 함수
- 확률 실험을 한 번 수행하면($\omega$를 뽑으면) 확률변수 X의 값 $x = X(\omega)$ 가 하나 결정됨
- 확률변수 자체에는 확률이 없음 — P는 $\Omega$ 위에 정의되고, X를 통해 $\mathbb{R}$로 '밀려 내려오는' 것

> **한 줄 요약**: X는 실험 결과($\omega$)를 숫자로 바꿔주는 함수 — $\omega$가 랜덤이라 X도 랜덤

**예시 — 동전 2회 던지기, X = 앞면 횟수**

| $\omega$ | HH | HT | TH | TT |
|---|---|---|---|---|
| $X(\omega)$ | 2 | 1 | 1 | 0 |

---

### 2-2. 이산 vs 연속 확률변수

| | 이산 (Discrete) | 연속 (Continuous) |
|---|---|---|
| 값의 범위 | 유한하거나 자연수/정수/유리수 집합 (셀 수 있음 — 값 사이에 빈틈 있음) | 실수 구간 (셀 수 없음 — 값 사이에 빈틈 없이 연속) |
| 수식 | $X : \Omega \to \{x_1, x_2, \ldots\}$ | $X : \Omega \to \mathbb{R}$ |
| 확률 부여 | 각 값에 양의 확률이 직접 부여됨 | $P(X=x) = 0$ → $P(a \leq X \leq b)$ 로만 계산 |
| 예시 | 주사위 눈, 이메일 스팸 여부, 단어 출현 횟수 | 키, 몸무게, 온도, 신경망 가중치 초기값 |

> **중요**: 연속 확률변수에서 $P(X = x) = 0$ → 반드시 구간으로 확률 계산: $P(a \leq X \leq b) = \int_a^b f(x)dx$

---

### 2-3. 누적 분포 함수 CDF (Cumulative Distribution Function)

```math
F_X(x) = P(X \leq x) \quad \text{이산 / 연속 모두 정의 가능한 가장 기본적인 분포 표현}
```

> $F(x)$ = x **이하**일 확률 → x가 커질수록 포함 범위가 넓어져 값이 누적됨

| 성질 | 수식 | 의미 | 예시 (주사위) |
|---|---|---|---|
| 단조 증가 | $x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)$ | 확률은 x가 커질수록 누적됨 | $F(1)=1/6,\ F(2)=2/6,\ F(3)=3/6$ |
| 경계값 | $\lim_{x \to -\infty} F(x) = 0,\ \lim_{x \to \infty} F(x) = 1$ | 전체 확률의 합은 1 | $F(0)=0,\ F(6)=1$ |
| 우연속 | $\lim_{\varepsilon \to 0^+} F(x+\varepsilon) = F(x)$ | 오른쪽에서 연속 | $F(2)=2/6$, 오른쪽$(2.0001)$→$F(2.0001)=2/6$ (같음), 왼쪽$(1.9999)$→$F(1.9999)=1/6$ (다름) |
| 구간 확률 | $P(a < X \leq b) = F(b) - F(a)$ | 두 CDF 값의 차이로 구간 확률 계산 | $P(2 < X \leq 4) = F(4)-F(2) = 4/6-2/6 = 2/6$ |

> CDF 장점: 이산·연속을 통일된 방식으로 표현. 이산의 경우 $P(X = x_0) = F(x_0) - F(x_0^-)$ (계단 함수의 점프 크기)

---

### 2-4. 확률 질량 함수 PMF (Probability Mass Function) — 이산

```math
p(x) = P(X = x) \quad \text{각 가능한 값 x에 직접 확률을 부여}
```

**PMF의 필수 조건**

- $p(x) \geq 0$
- $\sum_x p(x) = 1$
- $P(X \in A) = \sum_{x \in A} p(x)$

**예시 — 공정한 주사위**

| x | 1 | 2 | 3 | 4 | 5 | 6 |
|---|---|---|---|---|---|---|
| p(x) | 1/6 | 1/6 | 1/6 | 1/6 | 1/6 | 1/6 |

- CDF: $F_X(x) = \sum_{k \leq x} p(k)$ → x 이하인 모든 값의 확률을 더한 것
  - 예) $F_X(3) = p(1)+p(2)+p(3) = 1/6+1/6+1/6 = 3/6$
- 기댓값: $E[X] = \sum_x x \cdot p(x)$ → 각 값에 확률을 곱해서 모두 더한 것 (가중 평균)
  - 예) $E[X] = 1\cdot\frac{1}{6} + 2\cdot\frac{1}{6} + \cdots + 6\cdot\frac{1}{6} = 3.5$

---

### 2-5. 확률 밀도 함수 PDF (Probability Density Function) — 연속

```math
P(a \leq X \leq b) = \int_a^b f(x)dx \quad \text{구간에 대한 확률을 면적으로 계산}
```

**PDF의 필수 조건 & CDF와의 관계**

- $f(x) \geq 0$
- $\int_{-\infty}^{\infty} f(x)dx = 1$ (전체 면적 = 1)
- $F_X(x) = \int_{-\infty}^x f(t)dt$ (CDF = PDF의 적분)
- $f(x) = \frac{dF(x)}{dx}$ (PDF = CDF의 미분)

**PMF vs PDF 비교**

| | PMF $p(x)$ | PDF $f(x)$ |
|---|---|---|
| 대상 | 이산 확률변수 | 연속 확률변수 |
| 값의 의미 | $p(x) = P(X = x)$ 직접 확률 | $f(x)$ = 확률 밀도 (그 자체로 확률 아님) |
| 값의 범위 | $0 \leq p(x) \leq 1$ | $f(x) \geq 0$ 이지만 1보다 클 수 있음 |
| 전체 합/적분 | $\sum p(x) = 1$ | $\int f(x)dx = 1$ |

---

## 3. 결합, 조건부, 독립

### 3-1. 결합 확률 (Joint Probability)

```math
p(x, y) = P(X = x,\ Y = y) \quad \text{두 확률변수가 동시에 특정 값을 가질 확률}
```

- **두 사건이 동시에 일어날 확률**
- 예) "오늘 비가 오고 **동시에** 길이 막힐 확률" → 각각의 확률이 아니라 **둘 다** 일어나는 확률

**예시 — 동전(X) 앞/뒤, 주사위(Y) 1~6 동시에 던지기**

- $p(X=앞,\ Y=3)$ = 동전 앞면이면서 동시에 주사위 3이 나올 확률
- $= \frac{1}{2} \times \frac{1}{6} = \frac{1}{12}$

| | 이산 결합 PMF | 연속 결합 PDF |
|---|---|---|
| 조건 | $p(x,y) \geq 0$, $\sum_x \sum_y p(x,y) = 1$ | $f(x,y) \geq 0$, $\int\int f(x,y)dxdy = 1$ |
| 구간 확률 | - | $P(a \leq X \leq b, c \leq Y \leq d) = \int_c^d \int_a^b f(x,y)dxdy$ |

---

### 3-2. 주변 확률 (Marginal Probability)

> **한 줄 요약**: 결합 확률 표에서 한 변수를 무시하고 나머지만 묶어서 합친 확률 → 행/열의 합계

> 결합 분포에서 나머지 변수를 합산(이산)/적분(연속)해서 소거 → 단일 변수의 분포

**예시 — 동전(X) × 주사위(Y)**

| $p(x,y)$ | Y=1 | Y=2 | ... | Y=6 | **X 주변확률** |
|---|---|---|---|---|---|
| X=앞 | 1/12 | 1/12 | ... | 1/12 | **1/2** |
| X=뒤 | 1/12 | 1/12 | ... | 1/12 | **1/2** |

- 오른쪽 끝 합계 → X의 주변 확률 (Y 무시)
- 아래쪽 합계 → Y의 주변 확률 (X 무시)

| | 이산 | 연속 |
|---|---|---|
| X의 주변 분포 | $p(x) = \sum_y p(x,y)$ | $f(x) = \int f(x,y)dy$ |
| Y의 주변 분포 | $p(y) = \sum_x p(x,y)$ | $f(y) = \int f(x,y)dx$ |

- **직관**: Y가 어떤 값이든 상관없이 X=x일 확률 → Y에 대해 '무관심하게' 통합
- **비유**: 엑셀 표에서 행(X값)별로 열(Y값)을 모두 더해 행 합계를 구하는 것

> ML 연결: VAE의 목표는 $p(x) = \int p(x|z)p(z)dz$ 최대화 — z를 적분해 소거하는 주변화(marginalization)

---

### 3-3. 조건부 확률 (Conditional Probability)

```math
P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0 \quad \text{B가 주어졌을 때 A의 확률}
```

- **직관**: 'B가 일어났다'는 정보를 알게 되면 표본 공간이 $\Omega$에서 B로 좁아짐 → 그 좁아진 공간에서 A가 차지하는 비율

**연쇄 법칙 (Chain Rule of Probability)**

```math
P(A \cap B) = P(A|B) \cdot P(B) = P(B|A) \cdot P(A)
```

일반화:

```math
P(x_1, x_2, \ldots, x_n) = P(x_1) \cdot P(x_2|x_1) \cdot P(x_3|x_1,x_2) \cdots P(x_n|x_1,\ldots,x_{n-1})
```

| | 수식 | 설명 |
|---|---|---|
| 확률변수 버전 | $p(x\|y) = p(x,y)/p(y)$ | y가 연속이면 p(y)>0 조건 필요 |
| 전확률 공식 | $P(A) = \sum_k P(A\|B_k) \cdot P(B_k)$ | $B_k$: $\Omega$의 분할 → Bayes 분모의 계산 근거 |

> ML 연결: 언어 모델(GPT)의 핵심: $P(w_1,w_2,\ldots,w_n) = \prod_t P(w_t|w_1,\ldots,w_{t-1})$ — 연쇄 법칙을 직접 모델링

---

### 3-4. 독립성 (Independence) & 조건부 독립

```math
P(A \cap B) = P(A) \cdot P(B) \Leftrightarrow P(A|B) = P(A) \Leftrightarrow P(B|A) = P(B)
```

**독립의 의미**

- B가 일어났다는 정보가 A의 확률을 전혀 바꾸지 않는다 → 두 사건이 서로 '무관'
- **독립 ≠ 배반**: 배반이면 오히려 종속 ($P(A|B) = 0 \neq P(A)$)

**조건부 독립 (Conditional Independence)**

> Z라는 조건이 주어졌을 때 X와 Y가 **독립처럼 행동**하는 성질 (연쇄 법칙과는 별개 — 연쇄 법칙은 계산 도구, 조건부 독립은 변수 간 관계를 설명)

```math
X \perp Y | Z \Leftrightarrow P(X,Y|Z) = P(X|Z) \cdot P(Y|Z) \Leftrightarrow P(X|Y,Z) = P(X|Z)
```

- Z를 알고 나면 X와 Y가 독립이 됨 — Z가 X,Y의 공통 원인일 때 자주 등장
- 독립이지만 조건부 독립이 아닌 경우도 있고, 반대도 성립 → **독립과 조건부 독립은 다른 개념**

| 독립 가정 | ML 활용 |
|---|---|
| i.i.d. 가정 | $X_1,\ldots,X_n \sim p(x)$ i.i.d. → MLE의 $L(\theta) = \prod p(x_i|\theta)$ 분해 근거 |
| Naive Bayes 독립 가정 | $P(x|y) = \prod P(x_i|y)$ → 특징 간 조건부 독립 가정으로 계산 단순화 |

### 3-5. 베이즈 정리 (Bayes' Theorem)

- **새로운 데이터를 보고 나서 믿음을 업데이트하는 공식**
- 예) 처음엔 "비 올 확률 30%" → 하늘이 흐림을 보고 나서 → "비 올 확률 49%"로 업데이트

> 사전 믿음 $P(\theta)$ + 데이터($X$) 관측 → 사후 믿음 $P(\theta|X)$ 로 업데이트

```math
P(\theta|X) = \frac{P(X|\theta) \cdot P(\theta)}{P(X)} = \frac{\text{Likelihood} \times \text{Prior}}{\text{Evidence}}
```

| 항 | 기호 | 의미 |
|---|---|---|
| 사전 확률 (Prior) | $P(\theta)$ | 데이터 관측 전 $\theta$에 대한 믿음 |
| 우도 (Likelihood) | $P(X|\theta)$ | $\theta$가 주어졌을 때 데이터의 확률 |
| 증거 (Evidence) | $P(X) = \int P(X|\theta)P(\theta)d\theta$ | 정규화 상수 |
| 사후 확률 (Posterior) | $P(\theta|X)$ | 데이터 관측 후 갱신된 믿음 |

**예시 — 비 올 확률**

- $P(\theta) = 0.3$ : 비 올 사전 확률 30%
- $P(X|\theta) = 0.9$ : 비가 오면 하늘이 흐릴 확률 90%
- $P(X|\theta^c) = 0.4$ : 비 안 오는데 흐릴 확률 40%

1단계 — $P(X)$ 계산 (전확률 공식):

$P(X) = P(X|\theta) \cdot P(\theta) + P(X|\theta^c) \cdot P(\theta^c)$

$= 0.9 \times 0.3 + 0.4 \times 0.7 = 0.27 + 0.28 = 0.55$

2단계 — 사후 확률 계산:

$P(\theta|X) = \frac{0.9 \times 0.3}{0.55} \approx 0.49$

> **해석**: 하늘이 흐림을 관측 → 비 올 확률 30% → **49%** 로 업데이트!

## 4. 기댓값과 분산

### 4-1. 기댓값 E[X] (Expectation / Mean)

- **확률변수 X의 평균값** — 실험을 무한히 반복했을 때 나올 값들의 평균
- 각 값에 **확률을 가중치**로 곱해서 더한 것 (확률로 가중 평균)

| | 이산 | 연속 |
|---|---|---|
| 수식 | $E[X] = \sum_x x \cdot p(x)$ | $E[X] = \int x \cdot f(x)dx$ |

**함수의 기댓값 E[g(X)]**

```math
E[g(X)] = \sum_x g(x) \cdot p(x) \quad \text{(이산)} \quad \text{또는} \quad E[g(X)] = \int g(x) \cdot f(x)dx \quad \text{(연속)}
```

**예시 — 공정한 주사위**

기댓값 공식: $E[X] = \sum_x x \cdot p(x)$ = 각 값 × 그 값이 나올 확률을 전부 더한 것

주사위는 1~6이 각각 $\frac{1}{6}$ 확률이니까:

| x | p(x) | $x \cdot p(x)$ |
|---|---|---|
| 1 | 1/6 | 1/6 |
| 2 | 1/6 | 2/6 |
| 3 | 1/6 | 3/6 |
| 4 | 1/6 | 4/6 |
| 5 | 1/6 | 5/6 |
| 6 | 1/6 | 6/6 |
| **합계** | 1 | **21/6 = 3.5** |

$E[X] = \frac{1+2+3+4+5+6}{6} = \frac{21}{6} = 3.5$

> 주사위 눈 중 어느 것도 3.5가 아님 — 기댓값은 '가장 자주 나오는 값'이 아니라 **'중심 무게'** (확률로 가중 평균한 값)

---

### 4-2. 기댓값의 중요 성질

| 성질 | 수식 | 의미 |
|---|---|---|
| 선형성 (Linearity) | $E[aX + bY + c] = aE[X] + bE[Y] + c$ | 분포 형태와 무관하게 항상 성립 |
| 독립일 때 곱의 기댓값 | $X \perp Y \Rightarrow E[XY] = E[X] \cdot E[Y]$ | 독립이 아니면 성립 안 함에 주의 |
| 상수 | $E[c] = c$ | 상수의 기댓값은 상수 자체 |
| 단조 기댓값 | $X \geq 0 \Rightarrow E[X] \geq 0$ | 비음수 확률변수의 기댓값은 비음수 |
| Jensen 부등식 | $f$ 가 볼록(convex) 함수이면 $f(E[X]) \leq E[f(X)]$ | 정보이론에서 KL 발산 ≥ 0 증명의 핵심 |

---

### 4-3. 분산 & 표준편차 (Variance & Standard Deviation)

```math
Var[X] = E[(X-\mu)^2] = E[X^2] - (E[X])^2, \quad \mu = E[X]
```

- **분산의 의미**: 확률변수가 평균으로부터 얼마나 '퍼져 있는가'를 나타내는 척도
- **표준편차**: $SD[X] = \sqrt{Var[X]}$ (단위를 원래 단위로 복원)

**계산 공식 유도**

```math
Var[X] = E[(X-\mu)^2] = E[X^2 - 2\mu X + \mu^2] = E[X^2] - 2\mu E[X] + \mu^2 = E[X^2] - \mu^2
```

**예시 — 공정한 주사위**

1단계 — $E[X^2]$ 계산:

| x | $x^2$ | $x^2 \cdot p(x)$ |
|---|---|---|
| 1 | 1 | 1/6 |
| 2 | 4 | 4/6 |
| 3 | 9 | 9/6 |
| 4 | 16 | 16/6 |
| 5 | 25 | 25/6 |
| 6 | 36 | 36/6 |
| **합계** | | **91/6 ≈ 15.17** |

2단계 — 분산 계산 ($E[X] = 3.5$):

$Var[X] = E[X^2] - (E[X])^2 = 15.17 - 3.5^2 = 15.17 - 12.25 = 2.92$

3단계 — 표준편차:

$SD[X] = \sqrt{2.92} \approx 1.71$

> **해석**: 주사위 눈은 평균 3.5에서 평균적으로 **±1.71** 정도 떨어져 있음

**분산의 주요 성질**

| 수식 | 의미 |
|---|---|
| $Var[c] = 0$ | 상수는 퍼짐이 없음 |
| $Var[aX + b] = a^2 \cdot Var[X]$ | 이동(+b)은 분산에 영향 없음, 스케일(a)은 제곱 |
| $X \perp Y$ 이면 $Var[X+Y] = Var[X] + Var[Y]$ | 독립일 때만 분산의 합산 가능 |
| $s^2 = \frac{1}{n-1}\sum(x_i - \bar{x})^2$ | 표본 분산: n-1로 나눔 → 불편 추정량 (Bessel 보정) |

## 핵심 개념 — 딥러닝 연결

| 개념 | 핵심 수식 | ML 연결 |
|---|---|---|
| 확률 공간 $(\Omega, F, P)$ | $P: F \to [0,1]$, Kolmogorov 공리 3가지 | 모든 확률 모델의 수학적 기초 |
| 확률변수 X | $X: \Omega \to \mathbb{R}$ (가측 함수) | 데이터 = $p_{data}$ 의 실현값 |

| 결합·주변·조건부 | $p(x,y),\ p(x) = \sum p(x,y),\ p(x|y) = p(x,y)/p(y)$ | 생성 모델·분류기·언어 모델 |
| 독립 / 조건부 독립 | $P(A \cap B) = P(A)P(B)$ / $X \perp Y \| Z$ | i.i.d. 가정, Naive Bayes |
| 베이즈 정리 | $P(\theta\|X) \propto P(X\|\theta) \cdot P(\theta)$ | MLE·MAP·베이지안 학습 |
| 기댓값 / 분산 | $E[X] = \int xf(x)dx$ / $Var = E[X^2] - E[X]^2$ | 손실 최적화·정규화 이론 |
