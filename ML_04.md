# 기계학습 — 5. 편미분 & Gradient

---

## 1. 편미분 (Partial Derivative)

- **편미분**: 여러 변수가 있을 때, 나머지는 상수로 고정하고 딱 하나의 변수에 대해서만 미분하는 것
- 표기: `d` 대신 `∂` (라운드 d) 사용

| 구분 | 일반 미분 `f(x)` | 편미분 `∂f/∂x` |
|------|-----------------|---------------|
| 변수 수 | x 하나 | 여러 개 |
| 방법 | df/dx로 미분 | 나머지는 묶어두고 x만 미분 |

### 직관적 비유 — 산의 높이 h(x, y)

- x방향 편미분 → 동쪽으로 걸을 때 기울기
- y방향 편미분 → 북쪽으로 걸을 때 기울기
- → **어느 방향이 가장 가파른가?**

> **기계학습 연결**: 손실함수 $J(\theta_0, \theta_1, \ldots)$는 파라미터가 여러 개 → 각 파라미터에 대해 편미분해서 어느 방향으로 업데이트할지 결정

---

## 2. 편미분 계산 규칙

**규칙**: 미분하는 변수 이외의 것은 전부 숫자(상수)처럼 취급한다. 일반 미분 규칙 그대로 적용.

### 예제 1 | $f(x, y) = 3x^2y + 2y^3$

```math
\frac{\partial f}{\partial x} = 6xy \quad \text{(y는 상수 → 3y가 계수)}
```

```math
\frac{\partial f}{\partial y} = 3x^2 + 6y^2 \quad \text{(x는 상수 → 3x²는 그대로)}
```

### 예제 2 | $f(x, y) = e^{xy}$

```math
\frac{\partial f}{\partial x} = y \cdot e^{xy} \quad \text{(chain rule, y는 상수)}
```

```math
\frac{\partial f}{\partial y} = x \cdot e^{xy} \quad \text{(chain rule, x는 상수)}
```

### 예제 3 | $f(x, y) = \log(x) + xy^2$

```math
\frac{\partial f}{\partial x} = \frac{1}{x} + y^2
```

```math
\frac{\partial f}{\partial y} = 2xy
```

> ⚠️ **자주 하는 실수**: $\partial/\partial x\ [xy^2]$에서 $y^2$을 그냥 미분하면 안 됨. x에 대한 미분이므로 $y^2$은 계수 → 결과는 $y^2$

## 3. 그래디언트 벡터 (Gradient Vector) $\nabla f$

- **그래디언트**: 모든 변수에 대한 편미분을 하나의 벡터로 묶은 것
- **의미**: 함수가 가장 가파르게 **증가**하는 방향을 가리킴

### 정의

$f(\theta_1, \theta_2, \ldots, \theta_n)$에 대해:

```math
\nabla f = \left[\frac{\partial f}{\partial \theta_1},\ \frac{\partial f}{\partial \theta_2},\ \ldots,\ \frac{\partial f}{\partial \theta_n}\right]^\top
```

각 원소 = 해당 변수 방향의 기울기

### 예시 | $J(\theta_1, \theta_2) = \theta_1^2 + 2\theta_2^2$

```math
\nabla J = \left[\frac{\partial J}{\partial \theta_1},\ \frac{\partial J}{\partial \theta_2}\right]^\top = [2\theta_1,\ 4\theta_2]^\top
```

### 방향 정리

| 방향 | 의미 |
|------|------|
| $\nabla f$ 방향 | 현재 위치에서 함수값이 가장 빠르게 **증가**하는 방향 |
| $-\nabla f$ 방향 | 가장 가파르게 **감소**하는 방향 |

### 경사하강법과 연결

```math
\theta \leftarrow \theta - \alpha \cdot \nabla J(\theta)
```

- 그래디언트 반대 방향으로 조금씩 이동 = **최솟값 탐색**

> **핵심 직관**: 경사하강법은 '현재 위치에서 그래디언트를 계산하고, 그 반대 방향으로 α만큼 이동'하는 과정을 반복하는 것

## 4. 연쇄 법칙 (Chain Rule)

- **연쇄 법칙**: 함수 안에 함수가 겹쳐 있을 때 미분하는 규칙
- 딥러닝 **역전파(Backpropagation)의 핵심 수학 도구**

### ① 1변수 연쇄 법칙

$y = f(u)$, $u = g(x)$ 일 때:

```math
\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}
```

**예)** $y = (3x+1)^2$ 이면 $u = 3x+1$, $y = u^2$

```math
\frac{dy}{dx} = 2u \cdot 3 = 6(3x+1)
```

### ② 다변수 연쇄 법칙 (기계학습에서 핵심!)

$L = f(\hat{y})$, $\hat{y} = \sigma(z)$, $z = \theta^\top x$ 일 때:

```math
\frac{\partial L}{\partial \theta} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \cdot \frac{\partial z}{\partial \theta}
```

| 항 | 의미 |
|----|------|
| $\partial L / \partial \hat{y}$ | loss가 예측값에 얼마나 민감? |
| $\partial \hat{y} / \partial z$ | $\sigma(z)$ 미분 $= \sigma(1-\sigma)$ |
| $\partial z / \partial \theta$ | $z = \theta^\top x$ → 결과는 $x$ |

> **역전파(Backpropagation)** = 연쇄 법칙을 출력층에서 입력층 방향으로 반복 적용하는 것  
> 각 층의 그래디언트를 계산할 때 앞 층에서 이미 계산된 $\partial L / \partial(\text{앞 출력})$을 재사용 → 효율적

---

## 5. 핵심 요약

| 개념 | 수식 | 기계학습에서의 역할 |
|------|------|--------------------|
| 편미분 $\partial f/\partial \theta_k$ | $\partial J / \partial \theta_k$ | 파라미터 $\theta_k$ 하나를 바꿀 때 손실 J가 얼마나 변하는가 |
| 그래디언트 $\nabla J$ | $[\partial J/\partial \theta_1 \ldots \partial J/\partial \theta_n]^\top$ | 모든 파라미터에 대한 기울기 벡터 → 이동 방향 결정 |
| 경사하강법 | $\theta \leftarrow \theta - \alpha \cdot \nabla J(\theta)$ | 그래디언트 반대 방향으로 이동해 J를 최소화 |
| 연쇄 법칙 | $\partial L/\partial \theta = \partial L/\partial \hat{y} \cdot \partial \hat{y}/\partial z \cdot \partial z/\partial \theta$ | 여러 층 합성 시 그래디언트 계산 → 역전파의 수학적 근거 |

### 전체 학습 흐름

```
모델 예측          손실 계산          그래디언트          파라미터 업데이트
ŷ = σ(θᵀx)  →  J = -log(ŷ)  →  ∇J 계산         →  θ ← θ - α∇J
                              (편미분 + 연쇄법칙)
```
