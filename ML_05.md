# 기계학습 — 6. 선형 회귀와 이진 분류

## 1. 선형 회귀 (Linear Regression)

- **회귀(Regression)**: 연속적인 수치(실수)를 예측하는 지도 학습 문제

| 예시 | 입력 x | 출력 y |
|------|--------|--------|
| 주택 가격 예측 | 면적·위치·방 수 | 가격 (만원) |
| 주가 예측 | 전일 종가·거래량 | 다음 날 종가 |
| 기온 예측 | 습도·기압·계절 | 최고 기온 (°C) |
| 전력 수요 예측 | 요일·시간대·온도 | 전력 소비 (MW) |

### 모델

```math
\hat{y} = \theta_0 + \theta_1 x \quad \rightarrow \quad \hat{y} = \theta^\top x \quad \text{(벡터 표현, bias 항 포함)}
```

- $\theta = [\theta_0,\ \theta_1]^\top$, $x = [1,\ x_1]$
- N개 샘플: $Y = N \times 1$, $X = N \times d$, $\theta = d \times 1$

> **학습 목표**: 데이터 $(x_1, y_1), \ldots, (x_n, y_n)$이 주어졌을 때, 예측값 $\hat{y}$와 실제값 $y$의 차이를 가장 작게 만드는 $\theta$를 찾는 것

---

## 2. MSE 비용 함수

- **비용 함수(Cost Function)**: 모델의 예측이 얼마나 나쁜가를 수치화

### 평균 제곱 오차 (MSE, Mean Squared Error)

```math
J(\theta) = \frac{1}{N} \sum_i (\hat{y}_i - y_i)^2 = \frac{1}{N} \sum_i (\theta^\top x_i - y_i)^2
```

| 이유 | 설명 |
|------|------|
| 왜 제곱? | 음수·양수 오차 상쇄 방지 / 미분 가능 → 경사하강법 적용 가능 |
| 왜 평균? | 데이터 수 N에 무관하게 스케일 일정 유지 |
| 기하학적 의미 | 회귀직선까지의 수직 거리 제곱합 최소화 = **최소제곱법** |

> **MAE vs MSE**: MAE = $(1/N)\sum|\hat{y}_i - y_i|$ — 이상치에 강인하지만 미분 불연속 지점 존재. MSE — 이상치에 민감(오류가 제곱으로 커짐), 미분 연속 → 최적화 용이. 딥러닝에서는 MSE가 기본.

```math
\theta^* = \underset{\theta}{\arg\min}\ J(\theta) = \underset{\theta}{\arg\min}\ \frac{1}{N} \sum (\theta^\top x_i - y_i)^2
```

## 3. 정규방정식 (Normal Equation)

MSE를 최소화하는 $\theta$를 미분을 이용해 **해석적(analytic)** 으로 구하는 방법

| 단계 | 내용 |
|------|------|
| Step 1 | 행렬 표현: $J(\theta) = (1/N)\|X\theta - y\|^2$ ($X$: $N \times d$ 설계 행렬, $y$: $N \times 1$ 레이블 벡터) |
| Step 2 | 미분: $\nabla_\theta J(\theta) = (2/N)X^\top X\theta - (2/N)X^\top y = 0 \rightarrow X^\top X\theta = X^\top y$ |
| Step 3 | **해**: $\theta = (X^\top X)^{-1} X^\top y$ ← 정규방정식 |

**장점** — 반복 학습 불필요, 하이퍼파라미터 없음, 소규모 데이터에 효율적

**단점** — $(X^\top X)$ 역행렬 계산 $O(d^3)$, $d$가 크면 느림 / $X^\top X$가 특이행렬이면 역행렬 불가 (d > N일 때 발생) → Ridge 정규화로 해결 / 딥러닝(비선형 모델)에는 적용 불가 → 경사하강법 사용

---

## 4. 경사하강법 (Gradient Descent)

정규방정식이 불가능한 경우(d가 크거나 비선형 모델)에 반복적으로 $\theta$를 업데이트해 최솟값을 찾는 방법

### 파라미터 업데이트 규칙

```math
\theta \leftarrow \theta - \alpha \cdot \nabla_\theta J(\theta) \qquad (\alpha: \text{학습률, learning rate})
```

### MSE에 대한 그래디언트

```math
\nabla_\theta J(\theta) = \frac{2}{N} X^\top (X\theta - y) \quad \rightarrow \quad \theta \leftarrow \theta - \frac{2\alpha}{N} X^\top (X\theta - y)
```

| 방식 | 사용 데이터 | 특징 |
|------|------------|------|
| 배치 경사하강법 (Batch GD) | 전체 N개 | 안정적이지만 느림, 대규모에 비효율 |
| 확률적 경사하강법 (SGD) | 1개 샘플 | 빠르지만 불안정(노이즈 큼), 지역 최솟값 탈출 가능 |
| 미니배치 경사하강법 (Mini-batch GD) | B개 (B=32~256) | 배치와 SGD의 절충안, ★ 딥러닝의 표준 방식 |

> ⚠️ 학습률 $\alpha$ 선택: 너무 크면 발산, 너무 작으면 학습이 매우 느림 → 학습률 스케줄링 또는 Adam·RMSProp 같은 적응적 최적화 기법 사용

## 5. 다항 회귀 & 과적합

- 비선형 관계도 **특징 변환**으로 선형 회귀 틀 안에서 표현 가능
- **다항 특징 변환**: $x \rightarrow [1, x, x^2, x^3, \ldots, x^m]$으로 확장 후 선형 회귀 적용

| 상태 | 설명 |
|------|------|
| 1차 (Underfitting) | 모델이 너무 단순, 데이터 패턴을 담지 못함 → High Bias |
| 적절한 차수 (Good Fit) | 훈련·테스트 오차 균형 |
| 고차 (Overfitting) | 훈련 데이터의 노이즈까지 학습 → High Variance |

> **해결책**: 교차 검증(Cross-Validation)으로 차수 선택 / 정규화(Regularization)로 계수를 제한

## 6. 선형 회귀 요약

| 항목 | 내용 |
|------|------|
| 모델 | $\hat{y} = \theta^\top x$ (또는 $\hat{y} = Wx + b$) |
| 손실 함수 | $J(\theta) = (1/N)\sum(\hat{y}_i - y_i)^2$ [MSE] |
| 최적화 ① | 정규방정식: $\theta = (X^\top X)^{-1}X^\top y$ (소규모 데이터) |
| 최적화 ② | 경사하강법: $\theta \leftarrow \theta - \alpha \cdot \nabla J(\theta)$ (대규모·딥러닝) |
| 신경망 연결 | 출력층 = 선형 회귀의 일반화: $\hat{y} = f(x; \theta)$ ($f$: 비선형 함수 구성) |

## 7. 이진 분류 (Binary Classification)

- **분류(Classification)**: 입력 x를 유한한 클래스 중 하나로 할당하는 지도 학습 문제
- 이진 분류: $y \in \{0, 1\}$

| 예시 | 클래스 1 | 클래스 0 |
|------|---------|---------|
| 스팸 메일 분류 | 스팸 | 정상 |
| 질병 진단 | 양성 | 음성 |
| 신용카드 부정거래 | 사기 | 정상 |

### 왜 MSE로 분류 문제를 풀면 안 되는가?

- 선형 회귀는 $\hat{y}$가 $[0, 1]$ 범위를 벗어날 수 있다 → '확률'로 해석 불가
- 이상치 1개가 결정 경계를 크게 왜곡시킨다 (MSE가 큰 오차를 제곱으로 과중 처벌)
- MSE는 오차 분포가 Gaussian임을 가정 → 이진 레이블 $\{0, 1\}$에는 부적합

---

## 8. 시그모이드 (Sigmoid) 함수

어떤 실수 $z$를 $[0, 1]$ 범위의 값으로 압축하는 함수 — **'확률'로 해석할 수 있게 해주는 함수**

```math
\sigma(z) = \frac{1}{1 + e^{-z}} \qquad \text{출력 범위: } (0, 1)
```

| 조건 | 출력 | 의미 |
|------|------|------|
| $z = 0$ | $\sigma = 0.5$ | 중립 |
| $z \rightarrow +\infty$ | $\sigma \rightarrow 1$ | $C_1$에 가까워짐 |
| $z \rightarrow -\infty$ | $\sigma \rightarrow 0$ | $C_0$에 가까워짐 |
| 미분 공식 | $\sigma'(z) = \sigma(z)(1 - \sigma(z))$ | ← 역전파에 핵심 |

**S자 곡선 값 참고**: $z=-2 \rightarrow \sigma \approx 0.12$, $z=0 \rightarrow \sigma=0.50$, $z=2 \rightarrow \sigma \approx 0.88$

### 로지스틱 회귀 모델

```math
P(y=1 \mid x;\theta) = \sigma(\theta^\top x) = \frac{1}{1 + \exp(-\theta^\top x)}
```

## 9. Cross-Entropy 손실 함수

로지스틱 회귀의 출력은 확률 → MSE 대신 **Cross-Entropy** 손실 사용

### 단일 샘플 손실 (Binary Cross-Entropy)

```math
L(\hat{y}, y) = -[y \cdot \log(\hat{y}) + (1-y) \cdot \log(1-\hat{y})]
\qquad (\hat{y} = \sigma(\theta^\top x) = P(y=1|x))
```

| 케이스 | 손실 | 동작 |
|--------|------|------|
| $y=1$ | $L = -\log(\hat{y})$ | $\hat{y}=1$이면 $L=0$, $\hat{y} \rightarrow 0$이면 $L \rightarrow \infty$ |
| $y=0$ | $L = -\log(1-\hat{y})$ | $\hat{y}=0$이면 $L=0$, $\hat{y} \rightarrow 1$이면 $L \rightarrow \infty$ |

### 전체 데이터셋 손실 (N개 샘플 평균)

```math
J(\theta) = -\frac{1}{N} \sum_i \left[ y_i \cdot \log(\hat{y}_i) + (1-y_i) \cdot \log(1-\hat{y}_i) \right]
```

> Cross-Entropy 손실 최소화 = Bernoulli 분포에 대한 **MLE(최대 우도 추정)** 와 수학적으로 완전히 동치

---

## 10. 로지스틱 회귀 — 경사하강법 학습

| 단계 | 내용 |
|------|------|
| Step 1 | $J(\theta) = -(1/N)\sum[y \cdot \log(\sigma(z)) + (1-y)\cdot\log(1-\sigma(z))]$, $z = \theta^\top x$ |
| Step 2 | $\partial J/\partial z = \sigma(z) - y$ (시그모이드 미분 성질 $\sigma'(z) = \sigma(z)(1-\sigma(z))$ 이용) |
| Step 3 | $\partial J/\partial \theta = (\sigma(z)-y) \cdot x \rightarrow (1/N)X^\top(\hat{y}-y)$ ← MSE Grad와 형태 동일! |

### 파라미터 업데이트

```math
\theta \leftarrow \theta - \frac{\alpha}{N} \cdot X^\top(\hat{y} - y) \qquad (\hat{y} = \sigma(X\theta))
```

> **로지스틱 회귀는 정규방정식 없음**: Sigmoid를 포함하므로 해석적 해가 없다 → 반드시 반복적 최적화 사용.  
> 단, Cross-Entropy + Sigmoid 조합은 볼록(convex) 함수 → **지역 최솟값 = 전역 최솟값**

## 11. 결정 경계 (Decision Boundary)

### 결정 규칙

```math
P(y=1|x) = \sigma(\theta^\top x) \geq 0.5 \rightarrow y=1 \text{로 예측}
```
```math
P(y=1|x) = \sigma(\theta^\top x) < 0.5 \rightarrow y=0 \text{으로 예측}
```

### 결정 경계 조건

```math
\sigma(\theta^\top x) = 0.5 \iff \theta^\top x = 0 \rightarrow \theta_0 + \theta_1 x_1 + \theta_2 x_2 = 0
```

### 2차원 예시

$\theta = [-6, 1, 1]$ 이면 결정 경계: $x_1 + x_2 = 6$ (직선)

| 영역 | 조건 |
|------|------|
| $y=1$ 영역 | $x_1 + x_2 > 6$ |
| 결정 경계 | $x_1 + x_2 = 6$ |
| $y=0$ 영역 | $x_1 + x_2 < 6$ |

> **비선형 결정 경계**: 입력 특징을 다항 변환하면 곡선 결정 경계 가능  
> 예) $z = \theta_0 + \theta_1 x_1^2 + \theta_2 x_2^2$ → 원형 결정 경계

## 12. 다중 클래스 확장 — Softmax

이진 분류(Sigmoid)를 K개 클래스로 일반화

```math
P(y=k \mid x; W) = \frac{\exp(w_k^\top x)}{\sum_j \exp(w_j^\top x)} \qquad \rightarrow K\text{개 클래스의 확률 합} = 1
```

| 비교 | Sigmoid (이진) | Softmax (다중) |
|------|---------------|---------------|
| 클래스 수 | $K=2$ | $K \geq 2$ |
| 출력 | $P(y=1|x) \in (0,1)$ (스칼라) | $[P(y=0), \ldots, P(y=K-1)]$ (벡터) |
| 관계 | $\text{Softmax}(K=2)$와 동치 | 확률 분포 형태 |

### 다중 클래스 Cross-Entropy 손실 (Categorical Cross-Entropy)

```math
J(W) = -\frac{1}{N} \sum_i \sum_k y_{ik} \cdot \log P(y=k | x_i; W) \qquad (y_{ik}: \text{one-hot 레이블})
```

> Categorical Cross-Entropy 최소화 = Categorical 분포에 대한 MLE

## 13. 정규화 (Regularization)

- **과적합(Overfitting)**: 훈련 데이터의 노이즈까지 학습 → 새로운 데이터에서 성능 저하
- **정규화**: 손실 함수에 패널티 항을 추가해 $\theta$가 과도하게 커지는 것을 제한

| 방식 | 수식 | 특징 |
|------|------|------|
| L2 정규화 (Ridge) | $J(\theta) = \text{원래 손실} + \lambda\|\theta\|^2$ | $\theta$를 0 방향으로 축소, 모든 특징 조금씩 사용 |
| L1 정규화 (Lasso) | $J(\theta) = \text{원래 손실} + \lambda\|\theta\|_1$ | 일부 $\theta$를 정확히 0으로 → 자동 특징 선택 (sparse) |

**정규화 강도 $\lambda$**:
- $\lambda = 0$: 정규화 없음
- $\lambda \rightarrow \infty$: $\theta \rightarrow 0$ (심한 underfitting)
- 교차 검증으로 최적 $\lambda$ 선택

> **MLE/MAP 연결**: L2 정규화 = Gaussian Prior를 사용한 MAP 추정 / L1 정규화 = Laplace Prior를 사용한 MAP 추정

---

## 14. 선형 회귀 vs 로지스틱 회귀 비교

| 비교 항목 | 선형 회귀 (Linear Reg.) | 로지스틱 회귀 (Logistic Reg.) |
|----------|------------------------|------------------------------|
| 문제 유형 | 회귀 (연속 출력) | 이진 분류 (확률 출력) |
| 출력 범위 | $\hat{y} \in \mathbb{R}$ (임의 실수) | $\hat{y} \in (0,1)$ (확률) |
| 출력 함수 | $\hat{y} = \theta^\top x$ (항등 함수) | $\hat{y} = \sigma(\theta^\top x)$ (Sigmoid) |
| 손실 함수 | MSE: $(1/N)\sum(\hat{y}-y)^2$ | Binary CE: $-(1/N)\sum[y\log\hat{y}+(1-y)\log(1-\hat{y})]$ |
| 해석적 해 | ✅ 정규방정식 $(X^\top X)^{-1}X^\top y$ | ❌ 반드시 경사하강법 |
| MLE 연결 | MSE = Gaussian MLE | Cross-Entropy = Bernoulli MLE |

> ★ 두 모델 모두 **선형 결정 경계**를 가진다. 비선형 경계가 필요하면 특징 변환 또는 신경망을 사용한다.
